### الخطوة 1: تهيئة قاعدة البيانات

1. اذهب إلى [Supabase Dashboard](https://app.supabase.com)

3. اذهب إلى **SQL Editor**
4. انسخ محتوى ملف `database_setup.sql` والصقه
5. انقر **Run** لتنفيذ جميع الأوامر

```env
VITE_PAYPAL_CLIENT_ID=ASkIL-MB5c23QEU
VITE_PAYPAL_MODE=live
PAYPAL_CLIENT_SECRET=EEWbuSsqqP9WMaxEd_7zeXeXbj1_9qMag5I_dfuaQOJLq1jTGQk_g-CnfeoDBLgCC5ZZ6_GT_3xsSGgp
```

### الخطوة 3: النشر على Vercel

1. ادفع الكود إلى GitHub
2. اذهب إلى [Vercel](https://vercel.com)
3. اختر **New Project**
4. اختر مستودعك
5. أضف متغيرات البيئة من ملف `.env`
6. انقر **Deploy**

### الخطوة 4: اختبار التطبيق

```bash

npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
```

## 🐛 استكشاف الأخطاء

### مشكلة: "Failed to connect to Supabase"
**الحل:**
- تحقق من `VITE_SUPABASE_URL` و `VITE_SUPABASE_ANON_KEY`
- تأكد من أن قاعدة البيانات نشطة
- تحقق من اتصال الإنترنت

### مشكلة: "PayPal authentication failed"
**الحل:**
- تحقق من `VITE_PAYPAL_CLIENT_ID` و `PAYPAL_CLIENT_SECRET`
- تأكد من استخدام Live API (ليس Sandbox)
- تحقق من أن المفاتيح صحيحة وغير منتهية الصلاحية

### مشكلة: "Withdrawal failed"
**الحل:**
- تأكد من أن بريد PayPal صحيح
- تحقق من أن الرصيد كافي
- تأكد من أن الحد الأدنى ($5) متوفر
